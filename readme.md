## Preparar el entorno

Ante todo, debemos tener instalado NODE JS.

```sh
git clone https://github.com/dnmartel/Coder-NodeJS.git
npm install

Las variables de entorno se cambian desde el archivo .env
ADMIN=true/false
PORT=8080
TIPO_PERSISTENCIA=memoria/json/mongo/firebase

```
