  • / - Raiz - Documentación

    • Setear rol en server.js ( Linea 7 ) - true / false

    Setear tipo de persistencia en /daos/index.js ( Linea 3 a 7 )

    /api/productos - API Productos
    • GET /:id? - Me permite listar todos los productos disponibles ó un producto por su id
    • POST / - Para incorporar productos al listado (disponible para administradores)
    • PUT /:id - Actualiza un producto por su id (disponible para administradores)
    • DELETE /:id - Borra un producto por su id (disponible para administradores)
    /api/carrito/ - API Carrito
    • POST / - Crea un carrito y devuelve su id
    • DELETE /:id - Vacía un carrito y lo elimina
    /api/carrito/:id/productos - API Carrito
    • GET / - Me permite listar todos los productos guardados en el carrito
    • POST / - Para incorporar productos al carrito por su id de producto
    • DELETE /:id - Eliminar un producto del carrito por su id de carrito y de producto
